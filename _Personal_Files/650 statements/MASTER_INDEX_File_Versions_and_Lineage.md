## FILE NAMING CONVENTION (Going Forward)

**Format:** `vN_CATEGORY_description-SUPERSEDES.ext`

**Where:**
- `vN` = Version number (v1, v2, v3, v4)
- `CATEGORY` = What type of file (HEGEMONY_GRID, ANALYSIS, PROTOCOL, etc.)
- `description` = Brief description of contents
- `SUPERSEDES` = What it replaces (optional, or FINAL if latest)

**Examples:**
- `v4_HEGEMONY_GRID_650-Political-Spectrum-FINAL.jsx` ← Current grid
- `v1_ANALYSIS_50-Statements-Blind-Generation.md` ← First analysis
- `v2_STATEMENTS_100-Additional-Diverse.md` ← Statement expansion
