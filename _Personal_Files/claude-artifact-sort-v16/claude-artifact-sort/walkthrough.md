# Unified Artifact Sorter Walkthrough - Gemini UI Refinement

I have refined the Gemini sidebar toolbar to improve clarity and usability.

## UI Improvements

### 1. Renamed "Summarise Chat"
- **Clarity**: The "Summarise" button has been renamed to **Summarise Chat** to better reflect that it processes the entire conversation's canvases.

### 2. Button Tooltips
- **Guided UX**: Added descriptive tooltips to all buttons in the Gemini sidebar toolbar:
  - **Summarise Chat**: "Generate summaries for all canvases in this chat"
  - **Inject**: "Inject prompt to generate or update artifacts"
  - **Refresh (↺)**: "Force rescan of canvases"
