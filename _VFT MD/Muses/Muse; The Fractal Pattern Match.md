This is how humans know anything really: pattern matching. We simply ask: Does object structure A match object B? And by how much? We measure this through the vectors of Why, Where, What, Who, Cause, and Effect.

Then we reverse engineer it. We ask if the proposed "reason why" for an unknown object matches the known possible cause. We can formalize this logic simply: If Object A is a 100/100 match for Object B, then A is B. If A is less than 100/100, A merely shares properties of B. If A is greater than 100/100, A is B plus something else. If A is 0/100, then A is categorically not B.

To be certain, we apply this check across the 7 Interrogatives (Q). For each Q, there are 7 sub-questions (q), and inside those, 7 components (c).

For something to truly be something else, it needs to be a perfect match—A(Q,q,c) must equal B(Q,q,c)—across all of those fractal points.
