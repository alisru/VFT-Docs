# Semantic Genealogy of the 16 -isms

**Protocol:** Dual-Parent Virtue/Vice Decomposition
**Source:** Triadic Model of Reality
**Status:** WORKING DOCUMENT

---

## The Rule

Each of the 16 domains is born from exactly **2 of the 7 Classical Virtues** and corrupted by exactly **2 of the 7 Deadly Sins**. The 17th pair (Patience / Impatience) governs traversal, not content.

**The 7 Virtues:** Faith, Hope, Charity, Prudence, Justice, Fortitude, Temperance
**The 7 Vices:** Pride, Envy, Wrath, Sloth, Greed, Gluttony, Lust

|---|--------|-----------------|-------------------|---------------|-----------------|
| 1 | **Maths** | Prudence + Justice | **Order** | Pride + Sloth | **Chaos** |
| 2 | **Spirituality** | Faith + Hope | **Faith** | Pride + Wrath | **Nihilism** |
| 3 | **Religion** | Charity + Justice | **Charity** | Wrath + Envy | **Hatred** |
| 4 | **Intelligence** | Hope + Prudence | **Hope** | Sloth + Envy | **Despair** |

**Rationale:**
- *Maths* = Prudence provides the logic, Justice provides the balance. Pride says "I don't need to check" and Sloth refuses to build.
- *Spirituality* = Faith provides belief, Hope provides direction. Pride elevates ego above the divine, Wrath lashes out against meaning itself.
- *Religion* = Charity provides the love, Justice provides the moral structure. Wrath turns love to fury, Envy poisons the structure with comparison.
- *Intelligence* = Hope provides conviction, Prudence provides the synthesis. Sloth abandons effort, Envy resents others' capacity.
