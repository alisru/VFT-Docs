## PHASE 1: DATA COLLECTION ✓

### Birth Data Verification

```
□ Birth date verified (day, month, year)
□ Birth time verified (hour, minute)
□ Birth time source documented (birth certificate / estimate / rectified)
□ Timezone confirmed for birth location
□ Coordinates obtained (latitude, longitude)
□ Location spelling verified
□ Daylight saving time accounted for (if applicable)
```

**Red Flags:**
- ⚠ Birth time "around" or "approximately" → Note as estimate
- ⚠ Birth time unknown → Limit analysis to planetary positions only
- ⚠ Timezone unclear → Research historical timezone rules for location
