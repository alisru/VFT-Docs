Look for **Sun-Moon aspect**
- Square (90°) = Friction

- Conjunction (0°) = Fusion

### STEP 5: Fill ONE Template (1 min)
Open **TEMPLATE_01_PULSE_PROTOCOL.md**
Replace [PLACEHOLDERS] with your data

## ESSENTIAL FORMULAS

### Planet → Plane Mapping
```
Q1 = Sun (WHO/Identity)
Q2 = Saturn (WHERE/Boundaries)
Q3 = Jupiter (WHAT/Expansion)
Q4 = Venus (WHY/Values)
Q5 = Mercury (HOW/Logic)
Q6 = Mars (CAUSE/Action)
Q7 = Moon (EFFECT/Emotion)
```

### 343-Cube Node
```
[n1.n2.n3] where:
n1 = Plane (1-7)
n2 = Lens (1-7)
n3 = Metric (1-7)

Each n maps to: WHO/WHAT/WHERE/WHY/HOW/CAUSE/EFFECT

Example: [1.1.1] = "Who of Who of Who?" = Core identity
```

### Aspect → i-Layer
```
Conjunction (0°) = i1 (Historical Cause)
Sextile (60°) = i2 (Spiritual Effect)
Square (90°) = i3 (Physical Friction)
Trine (120°) = i2 (Spiritual Effect)
Opposition (180°) = i3 (Physical Polarity)
```
