```
Q1 - Meta-Physical (WHO) → Sun

Q4 - Lyrical (WHY) → Venus
Q5 - Logical (HOW) → Mercury
Q6 - Historical (CAUSE) → Mars
Q7 - Emotive (EFFECT) → Moon

### The 343-Cube [n1.n2.n3]

```
n1 = Primary Plane (1-7)
n2 = Interrogative Lens (1-7)
n3 = Measurement Metric (1-7)

Total nodes: 7 × 7 × 7 = 343

Each node asks ONE specific question
Structure is RIGID, not interpretive
```

### The 252 States of Being

```
6 Personal Will states × 6 Social Will states = 36 base states
36 base states × 7 Objective Frames = 252 total states

Tracks consciousness position through belief space
```

```
A₀ (Start) → Q (Questions) → A₁ (Logic Check) →

Bayesian iteration toward truth

## TROUBLESHOOTING

### Common Issues

**Issue: Birth time unknown**
```
Solution:
- Limit to planetary positions only
- Use solar houses as approximation
- Offer rectification service
- Note limitation clearly in analysis
```

**Issue: Conflicting interpretations**
```
Solution:
- This is normal - humans contain contradictions
- Identify "core" vs "coping" patterns
- Use Pulse Protocol to determine hierarchy
- Frame as integration challenge
```

**Issue: 343-cube questions don't make sense**
```
Solution:
- ALWAYS follow structure rigidly
- Don't "interpret" what seems logical
- Write n1.n2.n3 notation first
- Generate question from formula
- Verify against interrogative map
```

**Issue: Analysis feels overwhelming**
```
Solution:
- Start with priority nodes only
- Complete one system before moving to next
- Use templates to maintain consistency
- Set time limits per phase
- Remember 80/20 rule
```
