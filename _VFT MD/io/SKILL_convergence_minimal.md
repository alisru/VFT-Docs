# Convergence Test — Minimal Protocol

## The 7 Planes
Q1 WHO · Q2 WHAT · Q3 WHERE · Q4 WHY · Q5 HOW · Q6 CAUSE · Q7 EFFECT

Pass condition: node produces a real, observable relational connection. Fail = no connection. Never argue the claim — ask what it connects to.

## Evidence Standards (always all three)
[A] Stated ideal — what the actor claims
[B] Actions within context — what they actually do
[C] Objective ideal — what coherent first-principles would require

## Phase 1 — Structural Scan
For each Q1–Q7: PASS / FAIL / PARTIAL + one-line note.
Fail = cannot produce real relational connection to next plane.

## Phase 2 — Grid Coordinate
υ axis: +2 Everyone · +1 Others · 0 Neutral · −1 My group · −2 Only me
ψ axis: +2 Creating systemic value · +1 Proactive · 0 Neutral · −1 Passive · −2 Extracting/destroying
R_net = 1 ÷ (V_Q1 × V_Q2 × V_Q3 × V_Q4 × V_Q5 × V_Q6 × V_Q7) where V: 1.0 = coherent, →0 = distorted
Perceptual Inversion: if stated (υ,ψ) diverges from calculated — flag explicitly.

## Phase 3 — Source Integrity
ΔH = ‖Vector_Ideal − Vector_Action‖ · <0.3 Pass · 0.3–0.7 Conditional · ≥0.7 Fail
Tribal Gap: same standard applied to self as applied to others?

## Phase 4 — Forensic
Fake Maximiser: IF (capacity >> effort) AND (goal unsolved) THEN actor keeps problem alive.
Helxis: Bait beneficiary ≠ Switch beneficiary → extraction pattern confirmed.
